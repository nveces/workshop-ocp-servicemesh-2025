Please review the official documentation to install the latest versions of these tools on your laptop following the optimum procedure

It is also required to have a GitHub account in order to create some repositories during this tutorial.

Check the versions of the tools installed on your laptop using the following commands. For example: 
$ oc version
Client Version: 4.15.14
Kustomize Version: v5.0.4-0.20230601165947-6ce0bf390ce3
Server Version: 4.17.17
Kubernetes Version: v1.30.7
$ git -v
git version 2.48.1

Remember to configure your Git account with your name and email address after to fork the repositories. This is important because Git uses this information to identify the author of the commits you create. You can configure your Git account using the following commands:

git config --local user.name "<Your Name>"
git config --local user.email "<your email>"
